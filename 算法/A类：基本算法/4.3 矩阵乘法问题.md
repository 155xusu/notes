# 矩阵乘法

## 1 矩阵乘法-STRASSEN算法

### 问题描述


### 算法原理

算法的基本思想在于以增加加减法的次数来减少乘法次数：
用了7次n/2 x n/2 矩阵乘法和18次n/2 x n/2 矩阵的加法


### 算法过程
* 将矩阵分块
```
A = a11	a12 
    a21	a22           
B = b11	b12
    b21	b22
```
* 计算分块矩阵的乘法
```
d1 = (a11 + a22) (b11 + b22)
d2 = (a21 + a22) b11
d3 = a11 (b12 – b22)
d4 = a22 (b21 –b11)
d5 = (a11 + a12) b22
d6 = (a21 – a11) (b11 + b12)
d7 = (a12 – a22) ( b21 + b22)
```

* 将分块矩阵乘法进行合并


### 算法效率
```
T (n) = m					若n=1
	=7T (n/2) + 18(n/2)2 a	若n≥ 2
```
```
T (n) = mnlog7+6anlog7-6an2
即运行时间为Θ (nlog7)= O（n2.81）
```
