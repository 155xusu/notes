> 主要是利用位运算解决一些巧妙的问题。

## 1 基础

### 位运算

符号|说明
|----|----|
& | 按位与
\| | 按位或
^ | 按位异或
\~ | 按位取反
\>\> | 右移
<< | 左移

## 2 特殊性质
操作 | 性质
|-----| -----|
n & (n - 1)  | n中的最后一个1变成0
^ | 相同的数抑或运算等于零。不同的数抑或运算等于1
n/2 | 等价于 右移一位 n >> 1
n*2 | 等价于 左移一位 n << 1
 n % 2 |等价于 判断二进制最右一位值 n \& 1


## 3 常见算法

### 快速幂

```
double myPow(double x, int n) {
    //使用二进制方法，将幂转换成二进制。二进制每个位的权重就是可以递推计算，与二分法效果相同。
    long long N=n;
    if(N<0){
        x=1/x;
        N=-N;
    }
    double result=1;
    double temp=x;
    while(N!=0){
        if( N & 1)result=result*temp;
        temp *=temp;
        N= N>>1;
    }
    return result;
```

### 快速乘法

```
int quickMulti(int A, int B) {
    int ans = 0;
    while(B) {
        if (B & 1) ans += A;
        A <<= 1;
        B >>= 1
    }
    return ans;
}
```

### 快速加法

```
int add(int a,int b){
    cout<<(unsigned int)-1<<endl;
    while(b != 0) { // 当进位为 0 时跳出
        int c = ((unsigned int)(a & b)) << 1;  // c = 进位
        a ^= b; // a = 非进位和
        b = c; // b = 进位
    }
    return a;
}
```
