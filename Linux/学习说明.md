## 关于Linux命令行的是那种风格

* UNIX 风格，选项可以组合在一起，并且选项前必须有“-”连字符
```
ps -a
```
* BSD 风格，选项可以组合在一起，但是选项前不能有“-”连字符

```
ps a
```
* GNU 风格的长选项，选项前有两个“-”连字符

```
ps --all
```
