
数据结构分为

* 线性数据结构
* 树型数据结构
* 图型数据结构


C++中的容器分为（都是线性的）
* 顺序容器
  * array 数组
  * vector向量
  * list 链表
* 关联容器
  * map 映射
  * set 集合
* 容器适配器
  * stack 栈
  * queue 队列


Java中的容器分为（都是线性的）
* 集合collection
  * List
  * Queue
  * Map
  * Set
